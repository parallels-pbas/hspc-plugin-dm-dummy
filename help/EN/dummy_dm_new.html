To configure the plug-in, fill the form:

Field Name
Description
Domain lookup mode (for Registration) Select the domains availability mode for registration:
  • Always available. The Dummy plug-in will always behave as if the domain being registered is available (not taken).
  • Always unavailable. The Dummy plug-in will always consider a domain being registered as taken.
  • Use whois server. The Dummy plug-in will use the Whois server assigned by you to top level domains to check whether a domain being registered is available.
Domain lookup mode (for Transfer) Select the domains availability mode for transfer.
Always reject domains registration. Domains can be marked as 'Registered' only manually. You can check this box to cancel the automated registration of a domain and confirm a domain registration later, manually.
Always reject domains transfer. Domains can be marked as 'Registered' only manually. You can check this box to cancel the automated transfer of a domain and confirm a domain transfer later, manually.
Always report error on NS synchronization. Name servers will never be marked as 'Synchronized with the Registrar'. You can check this box to cancel the automated name servers synchronization, which makes mane servers synchronization not possible at all.
